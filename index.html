Uso de Bootstrap con CDN

Lenguajes III - TP N°2

Este trabajo práctico fue desarrollado utilizando HTML junto con Bootstrap mediante CDN. Se implementó un carousel en el encabezado con tres imágenes y navegación lateral.

Bootstrap permite diseñar páginas web modernas de forma sencilla, utilizando componentes predefinidos como botones, carruseles y estructuras de diseño.

El objetivo del trabajo fue comprender la integración de herramientas externas y mejorar la presentación visual del sitio web.
