Lenguajes III - Trabajo Práctico N°2

Este trabajo práctico fue realizado utilizando HTML y Bootstrap mediante CDN. En el encabezado se incorporó un carousel con tres imágenes, el cual permite desplazarse entre los distintos slides mediante botones laterales.

Bootstrap facilita el diseño de páginas web modernas y ordenadas, ya que brinda componentes ya preparados, como carruseles, botones, formularios, tarjetas y muchas otras herramientas visuales.

En este proyecto también se incluyó una sección central con información descriptiva y un footer con datos de contacto o referencia de la fuente utilizada, cumpliendo así con todos los requisitos solicitados en la consigna.
